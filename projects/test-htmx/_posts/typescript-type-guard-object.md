みなさんこんにちは、かじりです。今回はobjectのtypeguardについてです。

## instanceofを使おう

概要です。以前はobjectの型判定をするtypeというkeyを持たせていたが、classにすることによりinstanceofで条件分岐できるようになった。

以下はコード例です

before

```
type Obj = {
    type: 'a'
} | {
    type: 'b'
}
const obj: Obj = {type: 'a'}

if (obj.type === 'a') {
    ...
} else if (obj.type === 'b') {
    ...
} else {
    ...
}
```

after

```
class A {}

class B {}

const obj = new A()

if (obj instanceof A) {
    ...
} else if (obj instanceof B) {
    ...
} else {
    ...
}

```
